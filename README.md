# Composeon

A beautiful showcase of Lobe Icons with stunning linking posts.

## Overview

Composeon is a modern web application that demonstrates the elegant use of Lobe Icons in a clean, responsive interface. The project features a grid layout showcasing various popular service icons with hover effects and smooth transitions.
